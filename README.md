# Getting Started

## Step 1
Add it in your root build.gradle at the end of repositories:
```groovy
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
## Step 2
Add the dependency.
```
dependencies {
        implementation 'com.github.DoMondo:Fancier:Tag'
}
```
